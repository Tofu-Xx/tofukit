<h1 align=center>resetcss</h1>

### introduce

You don't want to reset your css every time

### use

```css
@import "https://unpkg.com/@tofukit/resetcss";
```
**or**
```html
<link rel="stylesheet" href="https://unpkg.com/@tofukit/resetcss">
```
